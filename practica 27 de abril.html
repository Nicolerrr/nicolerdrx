Objetivos de una pagina web

1Objetivo General

Diseñar y crear paginas web que incluyen diferentes elementos tales como fondo, textos en diferentes elementos tales como fondo, textos en distintos formatos, alineacion y colores, tablas, imagenes, menu de investigacion, entre otros

3Objetivo Especifico II

Identificar las propiedades de la tecnologia de HTML para colocar estilos a una pagina web.
